## Installation

```shell
$ npm install @zlicom/autoprefixer --save-dev
```

## Why @zlicom/autoprefixer ?

* 轻量级前缀添加工具集
* 支持自定义前缀
* 原生 API 代理

# Customize

```shell
  // src/prefixer.ts
  const PREFIX_STR = '/you_prefixer'
```

```shell
  $ pnpm build
  $ npm login
  $ pnpm publish
```
